# Story Code Quest

![Story Code Quest](https://raw.githubusercontent.com/parulnith/story-code-quest/main/public/assets/title_image.png)

Story Code Quest is a story-based programming puzzle game for children. Players arrange simple commands, run them step by step, and guide characters through familiar story worlds while learning sequencing, loops, debugging, and route planning.

Play the live game: [story-code-quest.vercel.app](https://story-code-quest.vercel.app/)

## What Kids Learn

- Sequencing actions in the right order
- Using repeat loops to simplify repeated steps
- Debugging commands when a route fails
- Avoiding obstacles and planning safe paths
- Connecting programming logic with story-based problem solving

## Gameplay

Each level has three main areas:

- A story panel that introduces the puzzle
- A canvas-based grid board where the character moves
- A command editor where players arrange commands

Available commands include:

- `move_forward`
- `turn_left`
- `turn_right`
- `pick_object`
- `drop_object`
- `repeat_loop`

## Levels

- The Thirsty Crow

## Run Locally

To run the project on your machine:

```bash
git clone https://github.com/parulnith/story-code-quest.git
cd story-code-quest
npm install
npm run dev
```

Then open the local URL shown by Vite, usually `http://127.0.0.1:5173/`.
